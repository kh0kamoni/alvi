Research Program

Research
Interests

My research focuses on understanding and mitigating security vulnerabilities across different layers of the computing stack - from kernel-level exploits to web application attacks. I employ both manual code analysis and automated techniques to discover and responsibly disclose vulnerabilities.

Focus Areas

Current Research Directions

Malware Analysis & Reverse Engineering

In-depth analysis of sophisticated malware families, including rootkits, ransomware, and APT tools. Development of behavioral signatures and detection methodologies.

Linux Kernel Security

Investigation of kernel vulnerabilities including use-after-free, race conditions, and privilege escalation vectors. Focus on eBPF security and kernel module analysis.

Web Application Security

Advanced persistent web attacks, novel XSS vectors, CSRF protections bypass, and deserialization vulnerabilities in modern web frameworks.

Exploit Development

Development of proof-of-concept exploits for complex vulnerabilities. Focus on bypassing modern exploit mitigations including ASLR, DEP, and CFG.

Vulnerability Discovery

Systematic approaches to vulnerability discovery through fuzzing, static analysis, and manual code auditing. Focus on critical open-source software.

ML for Security

Application of machine learning techniques for malware classification, anomaly detection, and automated vulnerability discovery.

Research Statement

Towards Automated Vulnerability Discovery

My research aims to develop novel techniques for automated vulnerability discovery across the software stack. By combining program analysis, fuzzing, and machine learning, I work to identify security flaws before they can be exploited in the wild. My work has led to the discovery of numerous CVEs in widely-used software, contributing to a safer digital ecosystem.

Current projects include developing specialized fuzzers for kernel modules, analyzing the security implications of eBPF, and creating datasets for training ML-based vulnerability detection models. I am particularly interested in vulnerabilities that cross privilege boundaries and affect core system components.
